# SayLess Store Listing

## Chrome Web Store Summary

Offline, local-first screen recording and transcript-based editing for developers.

## Chrome Web Store Description

SayLess records your screen, tab, region, or camera in Chrome and keeps the workflow on your device. Capture video, annotate while recording, reopen local recordings from the Videos tab, generate transcripts with the bundled Whisper model, edit by timeline or transcript selections, and export files you control.

Free to use. No signup required. All extension features are included, with no hidden access levels or separate capability modes.

SayLess is built for developer demos, bug reports, code walkthroughs, and local documentation workflows where recordings should remain portable and under user control.

Core capabilities:

- Screen, tab, region, and camera recording in the browser.
- Local recording library with rename, duplicate, import, export, delete, search, filters, thumbnails, and storage status.
- On-device transcription using packaged model assets.
- Transcript-driven timeline editing for cuts, mutes, captions, chapters, and local zoom moments.
- MP4, WebM, GIF, WAV, M4A, WebVTT, transcript JSON, and portable project sidecar exports.
- Recovery flows for interrupted local recordings and repairable missing-media states.
- Local diagnostic ZIP export for support and debugging.

## Privacy

Recordings, transcripts, project sidecars, thumbnails, and export jobs are handled locally in the browser storage and download surfaces available to Chrome. Release defaults use the bundled Whisper model and run transcription locally without sending recordings to external services.
