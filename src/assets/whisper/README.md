# Local Whisper Assets

SayLess release builds load Whisper from this extension-packaged asset tree.
Remote model downloads are disabled by default.

Expected layout:

```text
src/assets/whisper/models/onnx-community/whisper-base_timestamped/
  config.json
  generation_config.json
  preprocessor_config.json
  tokenizer.json
  tokenizer_config.json
  onnx/
    decoder_model_merged_quantized.onnx
    encoder_model_quantized.onnx
```

Run `npm run verify:whisper-assets` before packaging an offline release. The
large model files are intentionally not generated by the app at runtime.
